# Article Galaxy Widget

Article Galaxy Widget is a shopping-cart like overlay that works on PubMed, Google Scholar and more than 70 other discovery websites, including yours!

# Installation

  1. Add a button or link to your page so users can load Article Galaxy Widget. By default, Article Galaxy Widget discovers all of the bibliographic references on your page, so you may want to position your button in a sidebar or at both the top and bottom of search results. Or you could display multiple buttons inline with references and provide the index argument to make Article Galaxy Widget list only the single reference. Button examples:

Find all bibliographic references on the page and list them in the order encountered

<button onclick="loadArticleGalaxyWidget()">Access articles via Article Galaxy</button>

List only the single bibliographic reference at the provided zero-based index relative to all references found on the page. The following example would list only the 5th reference.

<button onclick="loadArticleGalaxyWidget(4)">Request article</button>
  1. Add this JavaScript just before the closing body tag on your page:
<script>
	function loadArticleGalaxyWidget(i) {
		var d = document, f = window.frames;
		if (f.length > 0 && 'BODY' !== d.activeElement.tagName) {
			d.activeElement.blur();
		}
		if (d.getElementById('rd-startup-bk')) {
			return;
		}
		var b = d.body;
		var a = d.createElement('div');
		a.innerHTML = '<div id="rd-startup-bk" data-embedded="true" data-item-index="'
				+ i
				+ '" style="position:fixed;z-index:2147483648;background:#0083FF;color:#FFF;font-size:18px;top:0;left:0;text-align:center;width:100%;padding:1em;font-family:arial;">Loading Article Galaxy Widget...</div>';
		b.appendChild(a);
		b.appendChild(d.createElement('script')).src = 'https://a-zlist.appspot.com/ordermultiple/ordermultiple.nocache.js';
	}
</script>

# Making your bibliographic references discoverable

Article Galaxy Widget uses multiple techniques to discover bibliographic references on your page. Give it a try using the installation instructions above. It may just work.

One way to ensure your references are discoverable is to add COinS markup (opens new window).

TIP

You can either wrap your visible reference markup with the span or just include the empty span anywhere on the page. Article Galaxy Widget will discover all spans on the page that have a Z3988 classname.

# Standalone Article Galaxy Widget Pages

If you prefer to open Article Galaxy Widget on it's own web page (as opposed to loading it as an overlay on your page), many options are supported. Just build a link with one of the following:

  • Comma-delimited pmids
  • Comma-delimited dois
  • A 'related_to_pmid' parameter with a single PMID, which will populate the widget with a list of ten related articles
  • Standard OpenURL parameters for a single reference

# pmids example:

http://app.reprintsdesk.com/ordermultiple?pmids=1234567,11149990

Try it (opens new window)

# dois example:

http://app.reprintsdesk.com/ordermultiple?dois=10.1378/chest.12-2963,10.1016/j.concog.2013.04.010,10.1371/journal.pone.0049228

Try it (opens new window)

# pmids and dois example:

http://app.reprintsdesk.com/ordermultiple?dois=10.1378/chest.12-2963,10.1016/j.concog.2013.04.010,10.1371/journal.pone.0049228&pmids=1234567,11149990

Try it (opens new window)

# related_to_pmid example:

http://app.reprintsdesk.com/ordermultiple?related_to_pmid=24351105

Try it (opens new window)

# OpenURL example:

http://app.reprintsdesk.com/openurl?ctx_enc=info%3Aofi%2Fenc%3AUTF-8&ctx_id=10_1&ctx_tim=2014-06-19T16%3A2%3A54EDT&ctx_ver=Z39.88-2004&res_id=http%3A%2F%2Fsfx.scholarsportal.info%2Fwindsor&rft.atitle=10%2C000+rare-disease+genomes+sequenced&rft.aulast=Nuala+Moran&rft.date=2014&rft.eissn=1546-1696&rft.epage=117&rft.genre=article&rft.issn=1087-0156&rft.issue=7&rft.jtitle=Nature+Biotechnology&rft.spage=114&rft.volume=32&rft_id=info%3Adoi%2F10.1038%2Fnbt0114-7&rft_val_fmt=info%3Aofi%2Ffmt%3Akev%3Amtx%3Ajournal&svc_val_fmt=info%3Aofi%2Ffmt%3Akev%3Amtx%3Asch_svc&url_ctx_fmt=info%3Aofi%2Ffmt%3Akev%3Amtx%3Actx&url_ver=Z39.88-2004
